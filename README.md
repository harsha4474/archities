# archities - Premium Interior Design Website

A cutting-edge, single-page React website for archities, a premium interior design business serving Andhra Pradesh and Telangana. Built with modern web technologies featuring 3D interactive elements, smooth animations, and a sophisticated user experience.

## üöÄ Features

- **3D Interactive Hero Section**: React Three Fiber powered 3D room visualization with floating particles and dynamic lighting
- **Responsive Design**: Fully responsive across all devices with Tailwind CSS mobile-first approach
- **Modern Animations**: Framer Motion powered scroll-triggered animations and micro-interactions
- **Glassmorphism UI**: Premium glass-morphic design elements with backdrop blur effects
- **Interactive Portfolio**: Masonry grid gallery with project details and modal views
- **Client Testimonials**: Auto-rotating testimonial carousel with smooth transitions
- **Contact Form**: Validated contact form with real-time feedback
- **SEO Optimized**: Complete meta tags and semantic HTML structure
